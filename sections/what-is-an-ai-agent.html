What Is an AI Agent?

An agent senses, reasons, plans, and acts toward explicit goals within a defined environment. Source

Key Insights

  • The PEAS model—Performance, Environment, Actuators, Sensors—remains the baseline for describing agent scope. Source
  • LLM “brains” rely on short-term context windows plus long-term memory via RAG to stay grounded and avoid hallucinations. Source
  • Sensors such as RAG connectors and actuators like MCP tool calls complete the perception-to-action loop. Source

Recommended Actions

  • Document each agent’s PEAS definition—including success metrics—before building. Source
  • Design explicit memory policies that pair conversation buffers with searchable knowledge stores. Source
  • List every tool an agent may invoke, the data it can touch, and the approval rules that govern actions. Source
